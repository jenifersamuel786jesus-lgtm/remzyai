## üöÄ Getting Started

### For Patients

1. **Sign Up**
   - Go to RemZy application
   - Click "Sign Up"
   - Enter username, email, password
   - Click "Create Account"

2. **Select Patient Mode**
   - After signup, you'll be asked to select device mode
   - Click "Patient Mode"

3. **Complete Patient Setup**
   - Enter your full name (required)
   - Enter date of birth (optional)
   - Enter safe area location (optional)
   - Click "Complete Setup"

4. **Save Your Linking Code**
   - You'll see an 8-character code (e.g., "1A4B53EA")
   - **IMPORTANT**: Write this down or take a screenshot
   - Share this code with your caregiver
   - Click "Finish"

5. **Start Using RemZy**
   - Face Recognition: Save people you know
   - AI Companion: Ask questions anytime
   - Tasks: Create reminders
   - Emergency: Use panic button if needed

---

### For Caregivers

1. **Sign Up**
   - Go to RemZy application
   - Click "Sign Up"
   - Enter username, email, password
   - Click "Create Account"

2. **Select Caregiver Mode**
   - After signup, you'll be asked to select device mode
   - Click "Caregiver Mode"

3. **Complete Caregiver Setup**
   - Enter your full name (required)
   - Enter phone number (optional)
   - Click "Complete Setup"

4. **Link to Patient**
   - Get the 8-character linking code from your patient
   - Enter the code in the "Link Patient" section
   - Click "Link Patient"
   - You'll now see the patient in your dashboard

5. **Monitor Patient**
   - View patient location
   - See task status
   - Check AI interactions
   - Receive alerts
   - View known faces

## üîç Troubleshooting

### "Failed to create patient profile"

**Solution**: 
- Check internet connection
- Sign out and sign in again
- Try again
- Check console logs for specific error

### "Patient not found" when linking

**Solution**:
- Verify linking code is correct (8 characters, uppercase)
- Patient must complete setup first
- Code is case-sensitive

### Face saving fails

**Solution**:
- Patient must complete setup first
- Check camera permissions
- Check internet connection
- Try again

### AI companion not responding

**Solution**:
- Patient must complete setup first
- Check internet connection
- Refresh page
- Try again
