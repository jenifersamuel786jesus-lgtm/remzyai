# Patient-Caregiver Linking - Quick Start

## For Patients

### Get Your Linking Code
1. Complete patient setup
2. You'll see an 8-character code (e.g., **A1B2C3D4**)
3. Share this code with your caregiver

## For Caregivers

### Link a Patient

**Option 1: During Setup**
1. Complete caregiver setup
2. Enter patient's linking code when prompted
3. Done!

**Option 2: After Setup**
1. Go to Dashboard
2. Click "Manage Patients"
3. Click "Link Patient" button
4. Enter the 8-character code
5. Click "Link Patient"

## Common Issues

### "Invalid linking code"
- Double-check the code (8 characters, no spaces)
- Make sure patient completed setup
- Try uppercase letters

### "Patient already linked"
- Check "Manage Patients" page
- Patient may already be in your list
- Refresh the page

### Patient not showing in list
- Refresh the page
- Log out and back in
- Check "Manage Patients" page

## Need Help?

See **PATIENT_CAREGIVER_LINKING_GUIDE.md** for detailed troubleshooting.

---

**Quick Reference**:
- Linking code: 8 characters (letters + numbers)
- Format: A1B2C3D4 (no spaces, no dashes)
- Case-insensitive (automatically uppercase)
- Never expires
- Can be reused for multiple caregivers
